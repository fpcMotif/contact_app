# **App Name**: Company Contacts

## Core Features:

- Employee Directory: Display a searchable list of company employees, sorted by department.
- Contact Detail: Display individual contact details, including name, department, phone number, and email address.
- Department Filter: Enable users to filter employees by department to quickly find the right person.

## Style Guidelines:

- Primary color: Light blue (#E0F7FA) for a clean and professional look.
- Secondary color: Gray (#607D8B) for text and subtle accents.
- Accent: Teal (#009688) for interactive elements and highlights.
- Clean and readable sans-serif font for names and contact information.
- Use a card-based layout for displaying employee information.
- Use simple, recognizable icons for phone and email actions.
